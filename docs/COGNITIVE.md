# V5 认知科学原理

> 从赫布理论到 V5 公式的完整推导——学习的数学本质

---

## 核心发现

V5 势垒方程不仅仅是记忆检索公式，它是**智能体的进化公式**。

```
学习 = 增加 Input (积累数据)
熟练 = 增加 γ (优化算法/降低延迟)
记住 = Input > B 且 γ 足够大，导致 P = 1 (逻辑门永续导通)
```

---

## 一、赫布理论 (Hebbian Theory)

### 经典表述

> "Cells that fire together, wire together."
> （一起放电的神经元，会连接在一起）

### 数学化表达

```
V5: "Input that fires together, gamma that wires together"
```

当输入信号(Input)频繁触发同一记忆回路时：
- 神经连接被强化
- 信号传递效率提升
- 最终形成条件反射

---

## 二、V5 公式的认知科学解读

### 公式回顾

```
P = 1 / (1 + e^(-2γ(Input - B)))
```

| 变量 | 认知科学含义 | 物理表现 |
|------|-------------|----------|
| **Input** | 输入强度/匹配度 | 复习次数、调用频率 |
| **γ (Gamma)** | 系统响应陡度 | 熟练程度、髓鞘化水平 |
| **B (Barrier)** | 激活阈值 | 神经激活门槛 |
| **P** | 激活概率 | 记忆导通程度 |

---

## 三、"越顺"的数学解释

### 1. Input = 复习

**初学阶段**：Input 极低（单词陌生、逻辑生疏）
- 此时 Input << B
- P → 0
- 系统处于"高阻态"，记忆无法导通

**复习过程**：每一次复习，都在增加 Input 的值
- 当 Input 不断增大，超过阈值 B
- 指数项 e^(-2γ(Input-B)) 迅速变小
- P 值沿着 S 曲线快速上升

> **物理意义**：复习不是简单重复，而是提高信噪比，把信号从噪声中"拉"出来。

### 2. γ = 熟练度/心流

**低 γ 状态（生涩期）**：
- Sigmoid 曲线平缓
- 即便 Input 达标，P 的变化也很慢（0.4 → 0.6）
- 需要消耗大量精力去"确认"记忆
- 反应迟钝

**高 γ 状态（熟练期）**：
- 曲线变得陡峭，接近阶跃函数
- 一旦 Input 稍微超过 B，P 瞬间从 0 跳变到 1
- **这就是"顺"！**
- 不需要思考，不需要检索
- 触发即响应

> 就像说母语，不需要在大脑里搜索词汇
> Input 刚产生，P 已经跳变到 1

## 四、认知闭环：从"复习"到"本能"

### 阶段一：认知建立

- **行为**：不断复习
- **数据变化**：Input 持续增加
- **状态**：P 值缓慢上升，记忆开始成型

### 阶段二：神经固化

- **行为**：高频使用，形成肌肉记忆
- **数据变化**：系统检测到高频调用，自动调高 γ 值
- **状态**：势垒变陡，记忆的导通速度极大提升

### 阶段三：本能反应

- **行为**：完全不假思索
- **数据变化**：Input 极高，γ 极高
- **状态**：P ≈ 1

> 此时，逻辑门完全打开，阻力为零。
> 这就是"顺"的极致。

```javascript
// 检测到高频调用 → 自动调高 γ
```

## 六、专家结论

### 学习的数学本质

```
学习 = 增加 Input (积累数据)
熟练 = 增加 γ (优化算法/降低延迟)
记住 = Input > B 且 γ 足够大 → P = 1
```

### V5 公式的双重身份

1. **技术公式**：记忆检索的概率计算
2. **认知公式**：智能体进化的数学表达

## 七、公式应用指南

### 如何"学会"一件事

1. **增加 Input**：反复练习、复习
2. **降低 B**：分解难度，阶梯式学习
3. **提高 γ**：专注、沉浸式训练

### 如何"不忘"

1. **保持 Input**：定期回顾
2. **维持 γ**：持续使用
3. **监控 P**：当 P < 0.5 时触发复习提醒
